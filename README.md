# WP AI Translator — Simple WordPress Plugin

A basic WordPress plugin that adds a custom button to translate any selected text into Hindi using AI APIs (can be extended further with Python backend or ML).

## Plugin Features
✔ Adds a Translate to Hindi button  
✔ Beginner-friendly clean PHP code  
✔ Safe WordPress functions

## How to Use
1. Download/clone the repo
2. Place the folder inside:
   `/wp-content/plugins/`
3. Activate plugin from WordPress dashboard

## Tech Used
- PHP
- WordPress
- Basic JavaScript
